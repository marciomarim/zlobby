# Elobby

Electron based lobby client

## Done

- Cross platform
- Auto-downloads engine, games and maps
- View minimaps before download
- Join, spectate and play
- Bubble chat
- Battleroom pm players
- Save, load and clear chats
- Notifications
- Account creation
- Watch replays
